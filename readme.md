# To-Do List

A simple, responsive To-Do List application . Users can add and remove tasks. Built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Fully responsive layout

## Project Structure
project/
├── index.html
├── styles.css
└── main.js
